IMPORTANT: CONTINUOUS OVERSIGHT & PARALLEL EXECUTION

Reviewers must conduct full-time tracking even if you are working on multiple files. Tracking, OST, and Metadata must be filled concurrently with the WFW translation.

Verification Steps

Extract sample sentences from translated files
Cross-check using back-translation (Google Translate)
Consult online dictionaries & terminology databases

MT Handling Protocol

1. STOP & FLAG 2. SEND BACK 3. ESCALATE

Unbreakable Rules

  • Never skip QC: Applies even for small 50-word tasks.
  • Zero MT Policy: Word-for-word copies will be rejected.
  • Independent Review: Reviewer ≠ Translator.

2. Translation Accuracy Check

"Verify excellence before the client sees it. Our internal QC is the final barrier against errors."

Reviewer Tip

For specialized files (Legal, Medical), always verify domain-specific terms using glossary checklists.

What to Verify

Meaning correct — no literal mismatch
No missing content / skipped segments
Numbers, names, dates accurate
Terminology fits project requirements

3. Reviewer Operations — Step-by-Step

1
File Preparation

Clean the file before upload. Fix layout formatting issues and remove manual breaks.

2
Project Setup in Phrase

Create the project in Phrase TMS and upload the prepared file to the workspace, attach TM, TB and run QA.

3
Linguistic QA Check & Verification

Review errors (tags, numbers, terminology). Run translation checks to identify any machine translation usage.

4
Approved for Handoff

Perform a final manual review of the exported file to ensure 100% quality and accuracy.
